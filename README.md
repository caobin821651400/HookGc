# HookGc
抑制GC

通过Native Hook ，应用冷启动的时候抑制GC来优化启动速度

在Android 10中Google系统自己也会在启动后抑制2秒，请见
https://cs.android.com/android/_/android/platform/art/+/a98a28262f645d100e2dee9587e7822d35ade6f9
